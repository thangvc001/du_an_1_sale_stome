SALES-STORM (QUẢN LÝ BÁN HÀNG THỜI TRANG)

> **Dự án:** Phần mềm quản lý cửa hàng quần áo (Desktop App)
> **Tác giả:** Vũ Chiến Thắng, Phạm Đức Toàn

## Giới thiệu
Ứng dụng Desktop giúp tối ưu quy trình bán hàng tại quầy, quản lý kho và nhân viên cho các shop thời trang. Giải quyết bài toán quản lý đa dạng thuộc tính sản phẩm (Màu, Size, Chất liệu).

## Tính năng nổi bật
- **Bán hàng tại quầy (POS):** Thao tác nhanh, in hóa đơn tự động.
- **Quản lý Sản phẩm:** Hỗ trợ biến thể chi tiết (Màu sắc, Kích thước, Chất liệu).
- **Quét mã QR:** Tìm kiếm sản phẩm và thanh toán cực nhanh bằng mã QR.
- **Quản lý Khuyến mãi:** Áp dụng Voucher giảm giá linh hoạt.
- **Thống kê:** Báo cáo doanh thu ngày/tháng và hàng tồn kho.

## Công nghệ sử dụng
- **Ngôn ngữ:** Java (Swing/AWT)
- **Database:** SQL Server
- **Thư viện:** JDBC, ZXing (QR Code), Apache POI (Xuất Excel)
- **IDE:** NetBeans
